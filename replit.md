# HOPE Heals San Antonio Website

## Overview
A professional, mobile-first website for HOPE Heals San Antonio, a nonprofit coalition dedicated to capacity building, supportive services, and advocacy for unsheltered populations in San Antonio.

## Project Structure
```
/
├── index.html          # Main HTML file with all sections
├── style.css           # Responsive CSS with mobile-first design
├── script.js           # JavaScript for interactivity
├── assets/
│   └── images/         # Image assets folder (placeholder for future photos)
└── replit.md           # This documentation file
```

## Brand Identity
- **Colors**: Deep Navy (#1a365d), Slate Gray (#64748b), Clean White
- **Fonts**: Merriweather (headings), Open Sans (body)
- **Aesthetic**: Faith & Community - authoritative, modern, compassionate

## Site Sections
1. **Home**: Hero with impact counter and CTA
2. **Our Story**: Organization history and values
3. **The Coalition**: Partnership and coordination focus
4. **Impact/Gallery**: Photo grid layout (placeholders)
5. **Get Involved**: Donors, Volunteers, Partners + Pledge form
6. **Contact**: Contact form with Google Workspace-ready footer

## Technical Details
- Mobile-first responsive design
- Sticky navigation with prominent DONATE button
- Animated impact counters
- Form submissions logged to console (ready for backend integration)
- SEO optimized metadata

## Language Guidelines
**DO USE**: Capacity Building, Supportive Services, Advocacy, Community Training
**DO NOT**: Claim housing provider status

## Running the Project
The website is served as static files using Python's built-in HTTP server on port 5000.
